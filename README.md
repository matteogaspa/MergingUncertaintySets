# Merging uncertainty sets via majority vote

This folder contains all the scripts needed to reproduce the results and plots in "Merging uncertainty sets via majority vote".


The `majorityVote.R` file contains the algorithm described in the Appendix. The other folders contain the simulations and analyses performed in the main text and in the Appendix.


Each folder contains the scripts to perfom the analyses, a `utils` file and (where applicable) the dataset.
